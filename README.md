# Next Maison

프리미엄 숙박 예약 플랫폼

## 기술 스택

- **Frontend**: Next.js 15, React 19, TypeScript
- **State Management**: TanStack Query, Zustand
- **Backend**: Supabase
- **UI**: Tailwind CSS, shadcn/ui
- **Deploy**: Vercel (예정)

## 주요 기능

- 숙소 검색 및 필터링
- 예약 시스템
- 실시간 댓글
- 찜하기 (위시리스트)
